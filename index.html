1. Introduction

This project is to make a website that presents the security handbook for client companies. This website includes the content of how to protect company data, networks and personal information. This website is necessary for non-technical companies.

Aim:

  • To create a simple static website (no need for a database or CMS).
  • To collect, list general and significant security problems, then point out its precautions and solutions.
  • Designing Readable UI.
  • Applying water model for project management.
  • Project must be released on 16th April 2020 (including design, develop, data entry, test, deploy).

Definition:

  • IT consultant company: help your company to solve IT problems. For example- Consultants from these companies will review your business needs and current operations. They then make recommendations and suggest new equipment and software. There are some fields like:
  • Cloud services
  • Disaster management
  • Network operations
  • Cybersecurity and compliance
